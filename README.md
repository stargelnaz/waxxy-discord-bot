Waxy Discord Bot

A serverless Discord bot for WAX blockchain interactions, deployed on AWS Lambda using the Serverless Framework.
This version is written in plain JavaScript (no TypeScript) and runs directly from the src/ folder.

Features

/wax – Get WAXP balance for a WAX wallet

/keiki – Get KEIKI token balance

Both commands default to the wallet: amfr2.wam unless a user provides one.
